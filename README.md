# Cоздание дерева DOM при помощи JS, загрузка файлов и сортировка на Drag and Drop

## Создание элементов DOM

Через Javascript создал форму для загрузки изображений, кастомный прогресс-бар при загрузке, кнопки отправки файлов на сервер и удаление конкретного файла из списка выведенных файлов. При превышении количества файлов больше 5 штук, выводится сообщение об ошибке. Также сообщение появаляется если размер файла больше 10 МБ.

## Drag and Drop

Реализовал Drag and Drop для загружаемого файла и сортировку файлов через Drag and Drop, путем добавления классов и атрибутов через JS.
